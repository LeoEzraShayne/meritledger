## 📖 关于《了凡四训》

《了凡四训》是明代袁了凡（袁黄，1533-1606）所著的家训，分为立命之学、改过之法、积善之方、谦德之效四篇。 袁了凡通过记录功过格的方式，改变了原本被命数注定的命运，实现了"命自我立，福自己求"的人生转变。

"命自我立，福自己求"

—— 袁了凡《了凡四训》

### 功过格的起源

功过格是一种传统的自我修身方法，通过记录每日的善行（功）和过失（过），培养自省习惯，提升个人修养。 袁了凡使用功过格三十余年，坚持不懈，终于改变了命运，并将这段经历写成《了凡四训》，传授子孙立命之道。

## 🎯 应用理念

功过格应用将这一传统修身方法数字化，让现代人也能通过每日记录功过，培养自省习惯，提升个人修养，改变自己的命运。

### 核心价值

- ✓ **自省修身：** 通过记录功过，培养自省习惯，认识自己的优点和不足
- ✓ **改过迁善：** 记录过失是为了改过，持续修行可以改变命运
- ✓ **传承智慧：** 将《了凡四训》的智慧传承到数字时代
- ✓ **多语言支持：** 支持中文、日文、英文，让不同文化背景的用户都能使用
